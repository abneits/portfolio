Projects

AI-Based Smart Playlist Generator

Unsatisfied with mainstream music recommendation algorithms, I developed an AI-driven system that creates highly personalized playlists based on daily context and user preferences. The system interacts with the Deezer API to push playlists, logs each played song in a MongoDB database, and checks playback history to avoid repetition. It also manages a personalized preference engine that evolves over time.

AI Wine Cellar Manager

This project manages my personal wine cellar using image recognition and AI. I take a photo of a wine bottle, and the system automatically retrieves detailed information about it, stores it in MongoDB, and tracks its aging process. Notifications are sent when a wine reaches ideal maturity, and an AI assistant recommends the best bottle pairing for a given meal based on the current cellar inventory.

Smart Daily Dashboard Frame

I built a digital photo frame that generates a personalized image every morning, combining various data sources: local weather, temperature sensors, garage door state, and other interactive home information. An AI model composes an aesthetic image integrating all this data, which updates every five minutes to reflect real-time changes in my environment.
